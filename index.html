About Sorting Visualizer

      
This visualization tool helps you understand how different sorting algorithms work by showing their step-by-step execution.

      

      
How to Use:

      
    
        
  • Click "New Array" to generate a random array
    • 
        
  • Choose a sorting algorithm from the options above
    • 
        
  • Adjust the array size and animation speed using the sliders
    • 
        
  • Click "Start Sorting" to see the algorithm in action
    • 
        
  • Watch the metrics to understand algorithm performance
    • 
        
  • Study the pseudocode in your preferred programming language
